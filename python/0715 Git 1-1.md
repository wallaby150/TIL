\- 원격저장소에 넣는 걸 push
\#설정 변경 : Repository default branch를 main에서 master로 바꿈.
\#설정 변경 : Create a new repository에서 repository 이름을 TIL로 만듦.


git remote add origin {https://github.com/wallaby150/TIL.git}
    \# 원격저장소를 추가할 건데, origin이라 부를 원격저장소의 위치는 여기야
git remote add second {~}
    \# 또 다른 원격저장소에도 같이 올려줘.
git push -u origin master
    \# 내가 지금까지 커밋한 것들 푸쉬할 건데, 내가 오리진이라고 이름지은 원격저장소에 마스터가 작업한 내용들을 푸쉬할 거야.

####Q.다른 곳에서 원격저장소의 것을 가져오는 법?
1. code - download zip하면 된다.
    그런데 이러면 master 안 뜬다. 즉 그냥 다운 받은 것 뿐이지.
2. 
```bash
$ git clone {https://github.com/wallaby150/TIL.git}
```
대신 폴더명은 레포지토리의 이름으로 될 것. 그건 다시 lecture처럼 바꾸면 됨.
    이러면 master 뜬다.

```bash
$ git pull origin master
```
하면 바뀐 내용이 있을 것이다.
